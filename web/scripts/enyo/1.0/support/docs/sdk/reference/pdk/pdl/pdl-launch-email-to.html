PDL_LaunchEmailTo

Description

Launches the email app with pre-defined (subject, value, recipients) values for a to-be-sent email.

Note: This API has been available since webOS 2.0 (PDK version 200).

Syntax 

PDL_Err PDL_LaunchEmailTo(const char *Subject, const char *Body,
  int numRecipients, const char **recipients)
Parameter Description
Subject Email subject.
Body Email body.
numRecipients Number of email recipients.
recipients Pointer to pointers to strings containing email addresses.

Returns 

PDL_NOERROR - Success
PDL_INVALIDINPUT - One of the parameters is null.
PDL_ECONNECTION  - Unable to communicate with app card.
