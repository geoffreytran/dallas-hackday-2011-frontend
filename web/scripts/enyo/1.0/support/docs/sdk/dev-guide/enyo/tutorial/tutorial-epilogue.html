Epilogue

Thanks for staying with us through the Enyo tutorial. There are at least two paths you can take from here.

The more bookish members of the audience may want to peruse the Enyo conceptual documentation. This collection of in-depth articles will boost your knowledge of how things work in Enyo and why they work that way.

Also, be sure check out the Enyo API documentation--it's commentary straight from the source (both literally and figuratively).

Those who prefer to learn by doing may want to add new features to the tutorial app. A few possible projects:

  • Make the code for parsing feed data more robust, so that it gracefully handles things like images.
  • Allow users to bookmark favorite feeds.
  • Allow users to choose different color schemes for the app.

Or, for extra credit, create an entirely new application.

Godspeed, Enyo Developers!
