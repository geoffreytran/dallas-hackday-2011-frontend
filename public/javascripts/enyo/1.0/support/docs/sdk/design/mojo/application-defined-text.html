Application-Defined Text

Applications have some scenes that include specific text for the user. There are three elements for providing application-defined text: body text, information text, and title.

The following table provides additional information with image examples.

Type Description Example
Body text This text is defined by the application and explains something in a scene, such as the text in the example. Implemented by using the palm-body-text style.
