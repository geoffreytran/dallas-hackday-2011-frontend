PDL_GesturesEnable

Description

Enables or disables the wave bar. Call this function with PDL_FALSE to disable; PDL_TRUE to enable. The default is enabled.

Syntax 

PDL_Err PDL_GesturesEnable(PDL_bool Enable);

Returns 

PDL_ECONNECTION — Unable to communicate with app card.
PDL_NOERROR — Success.

Note:

The wave bar allows you to access the Quick Launch menu from within fullscreen applications when you cannot access the standard launcher bar without first minimizing the app to Card view. It appears on screen as a wavy curve.


Device displaying the wave bar.
