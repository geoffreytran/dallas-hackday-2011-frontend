PDL_GetPendingPurchaseInfo

Gets status about a pending purchase. The orderNo field is returned in a PDL_PurchaseItem reply.

This call is a wrapper for a getPendingPurchaseInfo com.palm.service.payment call and returns the same response. Call PDL_GetItemReceiptJSON to turn the response into a JSON-formatted string.

Syntax 

PDL_ItemReceipt *PDL_GetPendingPurchaseInfo(const char *orderNo);

Parameters

Parameter
 Required Type Description
orderNo
 Yes Pointer Pointer to string containing order number from purchase call.

Returns

Pointer to PDL_ItemReceipt struct.
