PDL_BannerMessagesEnable

Description

Enables or disables banner messages. Banner messages do not stop the application, but pop up at the bottom of the screen and are displayed in a slow crawl within the Notification bar, which sits just below the application window in what is called negative space since it is outside of the card's window. Examples of banner messages include SMSes, and voicemail notifications. Call this function with PDL_FALSE to disable; PDL_TRUE to enable. The default is enabled.

Note:

Does not disable phone ringing or messages.

Syntax 

PDL_Err PDL_BannerMessagesEnable(PDL_bool Enable);

Returns 

PDL_ECONNECTION — Unable to communicate with app card.
PDL_NOERROR — Success.
