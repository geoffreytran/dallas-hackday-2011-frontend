PDL_GetCallingPath

Description

Deprecated. Just sets the passed buffer to './'.

Syntax 

PDL_Err PDL_GetCallingPath(char *buffer, int bufferLen);
Argument Description
buffer Pointer to the buffer.
bufferLen Length of the buffer.

Returns 

PDL_INVALIDINPUT — Passed buffer is NULL.
PDL_STRINGTOOSMALL — Length of passed buffer is less than 3.
PDL_NOERROR — Success.
