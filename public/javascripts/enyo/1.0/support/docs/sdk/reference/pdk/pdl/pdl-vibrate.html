PDL_Vibrate

Description

Vibrates the device.

Note: This API has been available since webOS 2.0 (PDK version 200).

Syntax 

PDL_Err PDL_Vibrate(int periodMS, int durationMS)
Parameter Description
periodMS Vibration frequency in milliseconds. If you want it to vibrate 50 times in one second, for example, you would set it to 20 (20ms x 50ms = 1000ms = 1 second). If this is zero, and durationMs is positive, it vibrates at maximum frequency.
durationMS Total period of vibration in milliseconds.

Returns 

PDL_INVALIDINPUT - periodMS or durationMS are negative numbers
PDL_NOERROR - Success or durationMS is 0
PDL_ECONNECTION - Unable to communicate with app card
